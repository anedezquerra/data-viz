Choosing the right module

DataViz is organized around analysis intent. Start from the question you need to answer, then choose the module that matches the data shape and workflow.

By Data Shape

One variable

Use Univariate for distributions, profiles, quality checks, outliers, weighted data, text, datetime, and bootstrap intervals.

Two variables

Use Bivariate for scatter, line, regression, category comparisons, agreement, lag, residual, and joint plots.

Many variables

Use Multivariate for heatmaps, pair plots, and parallel coordinates.

Process data

Use SPC for control charts, capability, rules, attributes, and process dashboards.

Static vs Interactive

Static functions return matplotlib axes or figures and are best for reports, notebooks, and publication workflows. Interactive functions return Plotly figures and are best for dashboards, exploratory review, and hover-driven inspection.

ax = dv.histogram(values)
fig = dv.histogram_interactive(values)
