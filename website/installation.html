DataVizInstallation

Install

Installation and development setup

Install the package and runtime dependencies before using the plotting APIs.

Runtime install

pip install dataviz

Local development

git clone https://github.com/anedezquerra/data-viz.git
cd data-viz
pip install -e ".[dev]"
pip install -r requirements-dev.txt

Runtime dependencies

The package uses matplotlib, seaborn, numpy, pandas, plotly, and scipy. Development also uses pytest, pytest-cov, black, flake8, mypy, sphinx, sphinx-rtd-theme, and kaleido.
