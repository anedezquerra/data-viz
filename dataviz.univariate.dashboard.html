dataviz.univariate.dashboard

Dashboard-style univariate analysis figures.

dataviz.univariate.dashboard.univariate_analysis_dashboard_interactive(data: Series | Sequence[float] | ndarray, bins: int = 30, title: str | None = None, color: str | None = None, template: str = 'plotly', height: int = 900, width: int = 1200) → Figure[source]

Create an interactive multi-panel univariate analysis dashboard.

Parameters:

  • data (SeriesLike) – Input observations.

  • bins (int) – Number of histogram bins.

  • title (Optional[str]) – Optional figure title.

  • color (Optional[str]) – Primary plot color.

  • template (str) – Plotly template name.

  • height (int) – Figure height in pixels.

  • width (int) – Figure width in pixels.

Returns:

Interactive univariate analysis dashboard.

Return type:

plotly.graph_objects.Figure

Raises:

  • TypeError – If values cannot be converted to numeric data.

  • ValueError – If bins is invalid or no observations remain.

Examples

`python fig = univariate_analysis_dashboard_interactive(data) `

Notes

The dashboard combines distribution shape, spread, normality, and robust-location diagnostics.

  • ValueError – If bins is invalid or no observations remain.

Examples

`python fig = univariate_analysis_dashboard_static(data) `

Notes
